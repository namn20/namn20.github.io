# 🤖 OpenCode: 당신의 AI 코딩 동반자

OpenCode는 터미널, IDE, 데스크톱 앱 등 다양한 환경에서 코드 작성을 돕는 오픈 소스 AI 코딩 에이전트입니다. 이 가이드는 OpenCode.ai를 설치하고 활용하는 방법을 안내합니다.

![OpenCode.ai](https://img.shields.io/badge/Tool-OpenCode.ai-8A2BE2)
![License](https://img.shields.io/badge/License-Open%20Source-green)

## ✨ OpenCode.ai란 무엇인가?

OpenCode.ai는 개발자가 코드를 더 빠르고 효율적으로 작성할 수 있도록 지원하는 AI 기반 도구입니다. 사용자의 코딩 환경에 통합되어 실시간으로 코드 제안, 버그 수정, 질문에 대한 답변 등을 제공합니다.

주요 특징은 다음과 같습니다.

-   **다양한 플랫폼 지원**: 터미널, VS Code와 같은 IDE, 그리고 별도의 데스크톱 애플리케이션에서 사용할 수 있습니다.
-   **광범위한 LLM 호환성**: 75개 이상의 LLM(거대 언어 모델) 제공자를 지원하며, Claude, GPT, Gemini 등 기존 AI 구독 서비스를 연결하여 사용할 수 있습니다. 로컬 모델 또한 지원합니다.
-   **개인정보 보호**: 사용자의 코드나 컨텍스트 데이터를 서버에 저장하지 않아 개인정보를 중요하게 생각하는 개발자에게 적합합니다.
-   **세션 관리**: 여러 코딩 세션을 동시에 관리하고, 동료와 세션을 공유하여 협업할 수 있습니다.
-   **LSP 연동**: 언어 서버 프로토콜(LSP)을 지원하여 더 정확하고 문맥에 맞는 코드 지원을 제공합니다.

## 🚀 설치 및 시작하기

OpenCode.ai는 간단한 `curl` 명령어를 통해 쉽게 설치할 수 있습니다.

### 설치

터미널을 열고 다음 명령어를 실행하세요.

```bash
curl -sSL https://opencode.ai/install.sh | sh
```

이 명령어는 OpenCode.ai 에이전트를 시스템에 다운로드하고 설치합니다.

### 초기 설정

설치가 완료되면, OpenCode는 사용자의 개발 환경에 맞게 몇 가지 초기 설정을 안내합니다. 주로 사용하는 LLM 서비스의 API 키를 연결하는 과정이 포함될 수 있습니다.

1.  터미널에서 `opencode` 명령어를 실행하여 에이전트를 시작합니다.
2.  브라우저 또는 터미널 UI를 통해 선호하는 LLM 모델을 선택하고 API 키를 설정합니다.

## 📖 기본 사용법

OpenCode.ai는 대화형 인터페이스를 통해 사용됩니다. 터미널이나 IDE에서 `opencode`를 활성화한 후, 자연어로 원하는 작업을 요청할 수 있습니다.

### 예시 1: 코드 생성하기

"Create a Python function that takes a list of integers and returns the sum of all even numbers."

```
opencode "Create a Python function that takes a list of integers and returns the sum of all even numbers."
```

OpenCode 에이전트가 해당 기능을 수행하는 Python 코드 블록을 생성해 줄 것입니다.

### 예시 2: 버그 수정하기

코드 파일을 열고 문제가 있는 부분을 선택한 후, OpenCode에 버그 수정을 요청할 수 있습니다.

"Fix the bug in this JavaScript code. The loop is not terminating correctly."

```
opencode "Fix the bug in this JavaScript code. The loop is not terminating correctly."
```

### 예시 3: 개념 질문하기

"Explain the difference between `let`, `const`, and `var` in JavaScript."

```
opencode "Explain the difference between `let`, `const`, and `var` in JavaScript."
```

## 🌟 고급 활용 팁

-   **IDE 통합**: VS Code 사용자의 경우, OpenCode 확장 프로그램을 설치하면 더욱 강력한 통합 기능을 경험할 수 있습니다. 코드 자동 완성, 실시간 디버깅 제안 등을 활용해 보세요.
-   **세션 공유**: 동료와 페어 프로그래밍을 할 때, `opencode session share` 명령어를 사용하여 현재 코딩 컨텍스트와 대화 내역을 공유할 수 있습니다.
-   **로컬 모델 활용**: 개인정보 보호가 매우 중요한 경우, 인터넷 연결 없이 로컬에서 실행되는 LLM을 설정하여 OpenCode와 함께 사용할 수 있습니다.

OpenCode.ai는 단순한 코드 자동 완성을 넘어, 개발의 전 과정을 함께하는 지능형 파트너입니다. 이 도구를 활용하여 생산성을 높이고 창의적인 문제 해결에 더 집중해 보세요.
